## Adjective Cheat Sheet (drill reference)

| Type | Example | て-form | before けど | modifying noun | negative | past negative |
|------|---------|---------|------------|----------------|----------|---------------|
| い-adj | 高い | 高**くて** | 高い**けど** | 高い + noun | 高**くない** | 高**くなかった** |
| い-adj (irregular) | いい | **よくて** | いい**けど** | いい + noun | **よくない** | **よくなかった** |
| な-adj | 便利 | 便利**で** | 便利**だ**けど | 便利**な** + noun | 便利**じゃない** | 便利**じゃなかった** |
| な-adj (trap) | きれい | きれい**で** | きれい**だ**けど | きれい**な** + noun | きれい**じゃない** | きれい**じゃなかった** |

> ⚠️ **The #1 reflex to build:** い-adj → drop い → add **くて**. Not いて, not て.
> ⚠️ **ません is for verbs only.** Adjective negatives use くない (い-adj) or じゃない (な-adj).
> ⚠️ **いい is irregular:** all forms use よ- (よくて, よかった, よくない) EXCEPT plain present (いい).

## Verb Negative Plain Form Reference

| う-row (dictionary ending) | あ-row + ない | Example |
|---|---|---|
| く | かない | 行く → 行かない |
| す | さない | 話す → 話さない |
| つ | たない | 待つ → 待たない |
| む | まない | 飲む → 飲まない |
| る | らない | 登る → 登らない |
| ぐ | がない | 泳ぐ → 泳がない |
| ぶ | ばない | 遊ぶ → 遊ばない |
| **う** | **わない** (exception) | 買う → 買わない |

> Group 2 (る-verbs): drop る, add ない (食べる → 食べない, 見る → 見ない)
> Group 3: する → しない, 来る → 来ない（こない）
