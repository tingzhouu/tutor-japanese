### 2026-02-21 — Lesson + Drill Session

**Source material:** Lesson transcript (weather, towns/places, Irodori Book 2 Ch.5 context)
**Drill topics:** は vs が particles · ている form · て/けど adjective connecting

#### Lesson errors (from transcript analysis)

- Particle confusion: は used where が needed for subjects/existence
- ている form: inconsistency between formal and casual forms (てんだ, てる vs ています)
- Verb phrase confusion: 髪の毛洗いを頭の上に置きたいです → 髪の毛を洗いたいです (word order/phrasing)
- でしょう overanalysis mid-sentence — processing lag under pressure

| Prompt | Given answer | Error | Correct |
|--------|-------------|-------|---------|
| "quiet and clean" | 綺麗い**で**静か | Added い to な-adj (きれい) | きれいで静かです |

| "famous and delicious" | 有名**て** | な-adj needs で not て | 有名**で** |
| "spacious, quiet, clean" | 広い**て**静かで | い-adj て: didn't drop い | 広**くて**静かで |
| "new café is..." | 新し**カフェ** | Dropped い from noun-modifying adj | 新し**い**カフェ |

#### What held up well

- Particle instincts mostly correct under pressure
- ている conjugation generally solid
- Natural use of と for "with"
- Past tense い-adjective (難しかった, 面白かった) — clean
- Self-corrected きれいい → きれいで mid-session without prompting
- Applied feedback: 便利だけど landed correctly after correction

#### Top priorities going into next session

1. **い-adj → くて reflex** — appeared twice in one drill; needs to be automatic
2. **な-adj before けど → だけど** — one miss, one correction
3. **きれい/有名 are な-adjectives** — they look like い-adj but aren't

### 2026-02-26 (Session 3) — Multi-particle Sentences (Scaled Difficulty)

**Goal:** Multi-blank sentences using full particle range: は/が/を/に/で/と/から/まで/の

#### Results

| Prompt | Errors | Correct |
|--------|--------|---------|
| 妹の誕生日___プレゼント___デパート___買いました | 誕生日**の**→**に** (occasion), プレゼント**は**→**を** (object) | 妹の誕生日**に**プレゼント**を**デパート**で**買いました |
| 兄___友達___公園___サッカー___しました | 友達**が**→**と** (nuance), 公園**に**→**で** (action location) | 兄**と**友達**と**公園**で**サッカー**を**しました |
| 家___学校___歩いて、教室___友達___日本語___話します | 家**に**→**から** (origin), 学校**を**→**まで/に** (endpoint) | 家**から**学校**まで**歩いて、教室**で**友達**と**日本語**で**話します |
| 駅___歩いて、ホーム___電車___待ちます | ホーム**は**→**で** (action), 電車**で**→**を** (object of 待つ) | 駅**に**歩いて、ホーム**で**電車**を**待ちます |

#### Error patterns this session

| Pattern | Error tendency | Correct |
|---------|---------------|---------|
| Time/occasion | using の instead of に | 誕生日**に**、月曜日**に** |
| Starting point | using に instead of から | 家**から** |
| Endpoint | using を instead of まで/に | 学校**まで** |
| 待つ object | using で instead of を | 電車**を**待つ |
| Action location | reaching for に when action happens there | 公園**で**サッカー |

#### New particle introduced
- **から** = from / starting point (家**から**、9時**から**、東京**から**)
- **まで** = until / up to (学校**まで**、5時**まで**、駅**まで**)
- **から〜まで** = from → to (家**から**学校**まで**)
